# Java-array-code

A  Java program that takes array elements as input, searches for a specific element, and finds the largest and smallest elements in the array using loops and conditional statements.

## Technologies Used
- Java
Scanner Class
Loops
Conditional Statements
Arrays

## Learning Objectives
Understand array declaration and initialization
Practice array traversal techniques
Implement searching and sorting logic
Work with matrices using 2D arrays
Improve problem-solving skills in Java
